# TechHub Retail Data Analytics Dashboard

A comprehensive Power BI dashboard for analyzing retail sales, customer behavior, and product performance at TechHub, an electronics retailer.

## 📊 Project Overview

This project presents an interactive Power BI dashboard designed to analyze retail business performance across revenue, profit, customers, and suppliers. The goal is to transform raw data into actionable insights that support strategic decision-making.

## 📁 Project Structure

```
techhub-retail-powerbi-dashboard/
├── Tech Hub Retail Data Analytics Dashboard.pbix    # Main Power BI file
├── Tech Hub Retail Data Analytics Dashboard.pdf     # Exported dashboard report
├── TechHub_Customers.csv                            # Customer data
├── TechHub_Products.csv                             # Product catalog
├── TechHub_Sales_Data.csv                           # Transaction data
└── README.md                                        # This file
```

## 🔧 Tools & Technologies

- **Power BI**: Interactive dashboard creation and visualization
- **DAX**: Data Analysis Expressions for calculated metrics and KPIs
- **Data Modeling**: Relational data model for integrated analysis
- **Data Source**: CSV files (Customers, Products, Sales Data)



## � Key Insights

- **Smartphones, Monitors, and Laptops** generate the highest revenue and profit
- **Nova Electronics and FutureTek** are top-performing suppliers
- **Customers aged 26–35** contribute the most revenue
- **Sales peak during November–December** and decline in mid-year

## 💡 Business Recommendations

- **Product Strategy**: Focus on high-margin products and optimize supplier selection
- **Customer Retention**: Improve retention strategies for low-value customer segments
- **Seasonal Planning**: Align marketing and inventory planning with seasonal demand patterns
- **Supplier Management**: Strengthen partnerships with top-performing suppliers and evaluate underperformers
